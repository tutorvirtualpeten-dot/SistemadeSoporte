Guía de Despliegue en la Nube (Gratis)

Esta guía te ayudará a subir el sistema Soporte Petén a internet utilizando servicios gratuitos de alta calidad.

Arquitectura:
Base de Datos: MongoDB Atlas
Backend: Render.com
Frontend: Vercel

Paso 1: Base de Datos (MongoDB Atlas)

  1. Ve a MongoDB Atlas y crea una cuenta gratuita.
  2. Crea un Cluster (elige la opción "FREE").
  3. En Database Access, crea un usuario (ej: admin) y una contraseña segura.
  4. En Network Access, agrega la IP 0.0.0.0/0 (permite acceso global).
  5. Ve a Database → Connect → Drivers.
  6. Copia tu Connection String. Se verá similar a:
    mongodb+srv://admin:password@cluster0.mongodb.net/?retryWrites=true&w=majority

Paso 2: Backend (Render)

  1. Crea una cuenta en render.com y conéctala con GitHub.
  2. Haz clic en New + y selecciona Web Service.
  3. Selecciona el repositorio SistemadeSoporte.
  4. Configura:
    • Root Directory: server
    • Build Command: npm install
    • Start Command: node index.js
  5. En Environment Variables, añade:
    • MONGODB_URI: (Tu cadena de conexión del Paso 1)
    • JWT_SECRET: (Una contraseña secreta cualquiera)
  6. Haz clic en Create Web Service. Copia la URL resultante (ej: https://api.onrender.com).

Paso 3: Frontend (Vercel)

  1. Crea una cuenta en vercel.com y conecta GitHub.
  2. Haz clic en Add New → Project.
  3. Importa SistemadeSoporte.
  4. En Root Directory, edita y selecciona la carpeta client.
  5. En Environment Variables, añade:
    • NEXT_PUBLIC_API_URL: https://tu-api.onrender.com/api
      (La URL de Render + /api al final)
  6. Haz clic en Deploy.

¡Listo!

Vercel te dará el enlace final de tu sitio web.
